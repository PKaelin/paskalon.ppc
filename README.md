# Power Plant Controller (PPC)
This project is not yet an actual power plant controller, but rather a simulation of one.  
It is designed to demonstrate the principles of control systems and how they can be applied to manage the operation of a power plant so as reference data for data analysis.


## Repository Structure
For easier navigation, the repository is organized into the following solution folders:
- paskalON.Ppc.Portfolio
- paskalON.Ppc.Sites
- paskalON.Ppc.OperatingModes
- paskalON.Ppc.PowerControls
- paskalON.Ppc.Devices
- paskalON.Common
